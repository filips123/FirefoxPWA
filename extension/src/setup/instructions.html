To install the program manually:
  1. Download and install the Visual C++ Redistributable.
  2. Download and run the PWAsForFirefox MSI installer.
  3. After the programs are installed, you should be automatically proceeded to the next step. If nothing changes after around 30 seconds, restart the browser.
Or... To install the package using Windows Package Manager:
  1. Make sure Windows Package Manager is installed.
  2. Install Visual C++ Redistributable using winget: winget install -e Microsoft.VCRedist.2015+..
  3. Install PWAsForFirefox using winget: winget install firefoxpwa --version
  4. After the packages are installed, you should be automatically proceeded to the next step. If nothing changes after around 30 seconds, restart the browser.
Or... To install the package using Chocolatey:
  1. Make sure Chocolatey is installed.
  2. Install the package using choco: choco install firefoxpwa --version
  3. After the package is installed, you should be automatically proceeded to the next step. If nothing changes after around 30 seconds, restart the browser.
Or... To install the package using Npackd:
  1. Make sure Npackd is installed.
  2. Install the package using ncl: ncl add -p firefoxpwa -v
  3. After the package is installed, you should be automatically proceeded to the next step. If nothing changes after around 30 seconds, restart the browser.
Or... To install the package using PortableApps.com:
  1. Make sure PortableApps.com Platform is installed.
  2. Download the PWAsForFirefox PAF package and install it using PortableApps.com Platform.
  3. Run the PWAsForFirefox program using PortableApps.com Platform and wait until a notification appears.
  4. When the program is running, you should be automatically proceeded to the next step. If nothing changes after around 30 seconds, restart the browser.
